# Proje Yönetim Sistemi

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş kapsamlı bir web uygulamasıdır. Kullanıcı dostu arayüzü ve performanslı çalışma yapısı ile modern standartlara uygun olarak kodlanmıştır.

## Kullanılan Teknolojiler

Proje geliştirme sürecinde aşağıdaki teknoloji yığını tercih edilmiştir:

- **Frontend Framework:** React
- **Build Tool:** Vite
- **Programlama Dili:** TypeScript
- **UI Kütüphanesi:** shadcn-ui
- **CSS Framework:** Tailwind CSS
